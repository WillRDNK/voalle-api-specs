# Especificações de APIs do ERP Voalle

Este diretório contém as especificações das rotas de API, seguindo um modelo padrão com um arquivo por rota.

## ⚠️ Informativo Importante
**As rotas, endpoints, parâmetros e estruturas descritos neste diretório são fictícios e servem apenas para expressar nossos desejos e requisitos de integração.** Eles devem ser adaptados conforme a documentação oficial e as APIs reais disponibilizadas pelo ERP Voalle e seus parceiros integrados.

## Estrutura
- `template.md`: Modelo padrão para criar novas especificações.
- `block-contract.md`: Rota para bloquear contrato.
- `unblock-contract.md`: Rota para desbloquear contrato.
- `create-event-type.md`: Rota para criar tipo de evento.
- `create-contract-event.md`: Rota para registrar evento em contrato.
- `payment-link-generation.md`: Especificação para geração de link de pagamento e webhook de confirmação.
- Outros arquivos para rotas futuras.

## Como Usar
Para adicionar uma nova rota, copie o `template.md`, renomeie e preencha com os detalhes específicos.
